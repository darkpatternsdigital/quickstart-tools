# DpdEmptySln

This is a basic .NET/pnpm starter.

- Leverages corepack for your `pnpm` runtime; ensure `corepack` is available on your path.
- Builds and lints via the `dotnet` cli.

## Build

Run the following command:

```sh
dotnet build
```
